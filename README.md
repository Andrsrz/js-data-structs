# js-data-structs
A small data structures library for JavaScript, also includes some algorithms like DFS and BFS.

## Implemented Data structures/ Algos-
- Trees
    - Binary Trees
    - Binary Search Trees
- Stack
- Queue
- Priority Queue
- Graph 
    - BFS
    - DFS
- Trie
- Power set (recursion method)
