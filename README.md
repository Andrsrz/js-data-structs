## Installing
### NPM package
`npm i js-data-structs`

[**NPM page**](https://www.npmjs.com/package/js-data-structs)

### CDN
`https://cdn.jsdelivr.net/npm/js-data-structs/DataStructures.js`

`https://cdn.jsdelivr.net/npm/js-data-structs/DataStructures.min.js`
